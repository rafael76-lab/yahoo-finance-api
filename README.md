# yahoo-finance-api
A dashboard was created with the purpose of enhancing my portfolio. In this case, it features an interactive chart built using real-time Apple stock price data retrieved via an API and the yfinance library.

Technologies: Python, pandas, Streamlit, yfinance, GitHub, Render.com

GitHub link: https://github.com/rafael76-lab/yahoo-finance-api
